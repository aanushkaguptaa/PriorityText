PriorityText API

Transform your text processing with our advanced dual-purpose API. Leveraging state-of-the-art machine learning, we offer precise spam detection with 98% accuracy and intelligent text summarization powered by Hugging Face T5.

Quick Start Guide

Make an API Request
{
    "text": "Your message here",
    "min_length": 10,    // Optional
    "max_length": 150    // Optional
}
Get Smart Results
{
    "result": "Ham",
    "summary": "Your text summary",

}

Code Examples

Python
import requests

response = requests.post(
    'https://prioritytext-api.onrender.com/predict',
    json={'text': 'Your message here'}
)

result = response.json()
print(f"Result: {result['result']}")
print(f"Summary: {result['summary']}")
JavaScript
fetch('https://prioritytext-api.onrender.com/predict', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({text: 'Your message here'})
})
.then(response => response.json())
.then(data => {
    console.log('Result:', data.result);
    console.log('Summary:', data.summary);
});
