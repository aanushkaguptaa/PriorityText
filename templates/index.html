Spam-No-More

This powerful spam detection API tool quickly and accurately flags unwanted content, providing essential protection against digital clutter.

Step-by-Step: How to Use the API

  1. Step 1: Send a POST request to the /predict endpoint with your message text. You’ll need to include a JSON payload like this:
    2. {
    "text": "Your message here"
}
  2. Step 2: The API will process your text, decide if it’s spam or not, and return a result.
  3. Step 3: The response will be a simple JSON indicating whether your message is Spam or Not Spam.

Example: Sending a Request

Here’s an example of how to use cURL to send a request to the API:

curl -X POST -H "Content-Type: application/json" -d '{"text": "You won a free vacation!"}' https://spam-no-more.onrender.com/predict

Response Format

If the text is spam, the API will respond with:

{
    "result": "Spam"
}

If the text is not spam, you’ll get:

{
    "result": "Not Spam"
}

Integrating the API in Your Projects

Want to use this API in your own project? It’s easy! Here’s a simple Python example to get you started:

import requests

url = 'https://spam-no-more.onrender.com/predict'
data = {'text': 'This is a test message.'}
response = requests.post(url, json=data)

print(response.json())

This snippet will send a POST request to the API and print whether the message is spam or not. You can integrate it with any messaging platform, email server, or even as part of a larger app.
