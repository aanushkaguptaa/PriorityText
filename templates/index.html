Spam-No-More

A powerful spam detection API tool that now includes a Hugging Face Summarizer to extract key insights from your input text!

Step-by-Step: How to Use the API

  1. Step 1: Send a POST request to the /predict endpoint with your message text. Include a JSON payload like this:
    2. {
    "text": "Your message here"
}
  2. Step 2: The API will process your text, determine if it’s spam, and also generate a brief summary using the Hugging Face model.
  3. Step 3: The response includes both spam detection results and the summary.

Example: Sending a Request

Here’s an example of how to use cURL to send a request to the API:

curl -X POST -H "Content-Type: application/json" -d '{"text": "You won a free vacation!"}' https://spam-no-more.onrender.com/predict

Response Format

If the text is spam, the API responds with:

{
    "result": "Spam",
    "summary": "Brief summary of the input text."
}

If the text is not spam, you’ll get:

{
    "result": "Not Spam",
    "summary": "Brief summary of the input text."
}

Integrating the API in Your Projects

Want to use this API in your own project? Here’s a simple Python example to get you started:

import requests

url = 'https://spam-no-more.onrender.com/predict'
data = {'text': 'This is a test message.'}
response = requests.post(url, json=data)

print(response.json())

This snippet sends a POST request to the API and prints whether the message is spam or not, along with its summary.
